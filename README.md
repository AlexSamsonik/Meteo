# Meteo
Get detailed information about the weather use polish recourse.

**Polish recourse:** https://www.meteo.pl/

How to use:
 - clone the repository;
 - install and activate virtualenv if necessary;
 - install requirement `(venv) $ pip install -r requirement.txt`
 - run main.py `(venv) $ python3 main.py -c city_name`
 - open /Meteo/city_name_date.pgn file (e.g. gdansk_2021_08_09.png)to get forecast weather.
 
Available city_name:
 - grodno
 - gdansk
